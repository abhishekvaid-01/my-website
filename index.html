How It Works

This visualizer demonstrates Heap Sort. The array is first organized into a max heap (complete binary tree where each parent node is larger than its children). Then, the largest element is repeatedly swapped to the end of the array and removed from the heap.

Node Colors:

  • Currently compared nodes
  • Swapped nodes
  • Sorted portion
  • Unsorted nodes
